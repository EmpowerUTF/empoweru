⚠️ Important Note on Light Mode

Light mode colors are provided for print materials, presentations, and accessibility needs. The primary digital experience should remain dark mode, as the brand's atmospheric gradients and premium aesthetic are designed for dark backgrounds.

✓ Correct Logo Usage

  • Always use "Empower.U" with period
  • Maintain letter spacing (0.12em)
  • Use appropriate gradient for background
  • Amber→Blue gradient adds warmth and variety
  • Keep adequate clear space around logo

✗ Never Do This

  • Don't write "EmpowerU" or "Empower U" (no space, needs period)
  • Don't use all lowercase "empower.u"
  • Don't stretch or distort the wordmark
  • Don't use decorative fonts
  • Don't use colors outside brand palette

✅ Do's & Don'ts

Color Usage

✓ DO

  • Use Royal Blue (#4C6FFF) on dark backgrounds
  • Use darker blue (#2563EB) on light backgrounds
  • Pair Royal Blue + Teal in gradients
  • Use Amber sparingly for accent gradients
  • Use improved secondary button (12% bg, 35% border)
  • Maintain contrast ratios above 4.5:1

✗ DON'T

  • Use bright Royal Blue on white (fails contrast)
  • Mix warm and cool gradients randomly
  • Use old dark secondary button (8% bg, 25% border)
  • Create new gradients without approval
  • Use neon or fluorescent variations
  • Ignore WCAG accessibility guidelines

Logo & Wordmark

✓ DO

  • Always write "Empower.U" with period
  • Maintain 0.12em letter spacing
  • Use weight 950 (Extra Black)
  • Choose gradient based on background
  • Use Amber→Blue gradient for variety
  • Provide adequate clear space

✗ DON'T

  • Write "EmpowerU" without period
  • Use lowercase "empower.u"
  • Stretch, rotate, or distort wordmark
  • Use decorative or script fonts
  • Place logo on busy backgrounds
